¿Qué es?

Ember.js se trata de un framework web que utiliza un patron de servicio de componentes el cual permite a los desarrolladores crear "single-page applications" utilizando lenguajes de programacion comunes, buenas practicas, y patrones de otros entornos de desarrollo de SPAs.

Filosofia y diseño

Ideas clave de diseño

·Concéntrese en aplicaciones web ambiciosas

Ember es un componente de un conjunto de herramientas que funcionan juntas para proporcionar una pila de desarrollo. Ember CLI proporciona una estructura de aplicación y una canalización de compilación con complementos.

·Estabilidad sin estancamiento

Esta es la idea de que la compatibilidad con versiones anteriores es importante y se puede mantener mientras se sigue innovando y evolucionando el marco.

·Más productivo desde el primer momento

·Previsión de futuros estándares web

Ember adoptó los estándares en torno a JavaScript y la web, incluidas las promesas, los componentes web y la sintaxis de ES6. Yehuda Katz, uno de los cofundadores de Ember, es miembro de TC39, que es el comité responsable de las futuras versiones del lenguaje JavaScript.
